Introducción a Pilas en Java

Definición de una Pila

Una pila es una estructura de datos que sigue el concepto LIFO (Last In, First Out), es decir, el último elemento en entrar es el primero en salir. Este tipo de estructura es útil en múltiples aplicaciones, especialmente en programación.

Operaciones Básicas de una Pila

  • push: Inserta un nuevo elemento en la cima de la pila.
  • pop: Elimina y devuelve el elemento de la cima de la pila.
  • peek: Muestra el elemento de la cima sin quitarlo.
  • isEmpty: Comprueba si la pila no tiene elementos.

Ejemplo de Implementación en Java

En Java, la clase Stack de la biblioteca estándar permite manejar pilas de manera sencilla. Aquí tienes un ejemplo básico:

import java.util.Stack;

public class EjemploPila {
    public static void main(String[] args) {
        Stack pila = new Stack<>();
        
        // Agregar elementos
        pila.push(1);
        pila.push(2);
        pila.push(3);
        
        // Mostrar el elemento en la cima
        System.out.println("Elemento en la cima: " + pila.peek());
        
        // Retirar el elemento
        System.out.println("Elemento retirado: " + pila.pop());
        
        // Verificar si la pila está vacía
        System.out.println("¿La pila está vacía? " + pila.isEmpty());
    }
}
